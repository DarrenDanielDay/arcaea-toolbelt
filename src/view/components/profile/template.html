b30相关的功能主要是手动录入成绩进行计算并生成看板，类似excel算分表格的在线版。

为了减少手动录入成绩的麻烦，下面也提供一个无需订阅 Arcaea Online 即可用的自助查分脚本工具，可用于在Arcaea官网爬取这些成绩数据。

所有的查分操作都会在浏览器内进行，数据也存在浏览器内。查分原理和 Arcaea Unlimited API 一样是查好友榜，使用此功能的封号风险请自行承担（建议注册小号加大号好友来查，无需互相关注）。

通常来说被查玩家的潜力值越低，查分越慢，最坏情况是查完将近1000个谱面的好友榜，大约需要5分钟。

在电脑上，使用Chrome/Edge浏览器打开登录官网，按F12打开开发者工具，找到控制台（Console），复制并执行以下脚本代码即可调用插件。

若需要频繁使用此脚本，可以使用油猴插件实现在访问Arcaea官网时自动执行。
